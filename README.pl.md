# Mailhilfe Order Note Manager for WooCommerce

## Wersja polska

Wersja 2.0.19 zawiera polski interfejs administracyjny, wbudowaną pomoc, najczęściej zadawane pytania i szablony demonstracyjne. Aby korzystać z polskiego interfejsu, ustaw język witryny lub język użytkownika WordPressa na polski.

## Najważniejsze funkcje

- Tworzenie i zarządzanie szablonami notatek zamówień WooCommerce
- Notatki wewnętrzne i notatki dla klientów
- Symbole zastępcze dla zamówień, klientów, płatności, wysyłki i produktów
- Kategorie, ulubione, wyszukiwanie, sortowanie i licznik użycia
- Import i eksport JSON oraz podgląd importu
- Warunki szablonów, centralna historia i dziennik przetwarzania wiadomości e-mail
- Podgląd z zamówieniem testowym, osobiste ulubione i ostatnio używane szablony
- Zgodność z WooCommerce HPOS

## Informacja dotycząca bezpieczeństwa

Notatki dla klientów mogą uruchamiać powiadomienia e-mail WooCommerce. Przed dodaniem notatki zawsze sprawdź podgląd, symbole zastępcze, dane osobowe oraz typ notatki.
