# Mailhilfe Order Note Manager for WooCommerce

WooCommerce ऑर्डर नोट्स के लिए पुनः उपयोग योग्य टेम्पलेट बनाने, व्यवस्थित करने और उपयोग करने वाला WordPress प्लगइन।

## हिंदी संस्करण

संस्करण 2.0.19 में इंटरफ़ेस, अंतर्निहित सहायता, अक्सर पूछे जाने वाले प्रश्न और डेमो टेम्पलेट का हिंदी अनुवाद शामिल है। हिंदी इंटरफ़ेस उपयोग करने के लिए WordPress में साइट या उपयोगकर्ता भाषा के रूप में हिंदी चुनें।

## मुख्य सुविधाएँ

- आंतरिक और ग्राहक नोट्स के लिए टेम्पलेट
- ऑर्डर, ग्राहक, शिपिंग और भुगतान डेटा के प्लेसहोल्डर
- नोट जोड़ने से पहले संपादन योग्य पूर्वावलोकन
- श्रेणियाँ, पसंदीदा, खोज और क्रमबद्धता
- टेम्पलेट शर्तें
- केंद्रीय इतिहास और ईमेल संसाधन लॉग
- पूर्वावलोकन सहित JSON आयात और निर्यात
- WooCommerce HPOS संगतता
- टेम्पलेट प्रबंधन और उपयोग के लिए अलग अनुमतियाँ

## इंस्टॉलेशन

1. `mailhilfe-order-note-manager` फ़ोल्डर को `/wp-content/plugins/` में अपलोड करें या WordPress एडमिन में ZIP फ़ाइल इंस्टॉल करें।
2. प्लगइन सक्रिय करें।
3. WordPress मेनू में **Mailhilfe ऑर्डर नोट्स** खोलें।
4. टेम्पलेट बनाएँ या डेमो टेम्पलेट इंस्टॉल करें।

## अनुवाद

रिपॉज़िटरी में सामान्य जर्मन, औपचारिक जर्मन, स्पेनिश, फ़्रेंच, इतालवी, हिंदी, रूसी और ब्राज़ीलियाई पुर्तगाली के बंडल कैटलॉग हैं। टेक्स्ट डोमेन `mailhilfe-order-note-manager` है। WordPress.org में प्रस्तुत करने से पहले हिंदी अनुवाद की अतिरिक्त मातृभाषी समीक्षा अनुशंसित है।
