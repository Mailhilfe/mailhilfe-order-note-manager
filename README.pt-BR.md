# Mailhilfe Order Note Manager for WooCommerce

O plugin fornece modelos reutilizáveis de notas de pedidos do WooCommerce, prévia de espaços reservados, notas internas e notas ao cliente, condições de modelo, histórico, diagnóstico e importação/exportação JSON.

## Tradução para português do Brasil

A versão 2.0.19 contém a tradução da interface, da ajuda integrada, das perguntas frequentes e dos modelos de demonstração para português do Brasil. Para usar a interface em português, selecione Português do Brasil como idioma do site ou do usuário no WordPress.

## Instalação

1. Envie o arquivo ZIP em “Plugins → Adicionar plugin → Enviar plugin”.
2. Ative o plugin.
3. Abra “Notas de pedido Mailhilfe” na área administrativa do WordPress.
