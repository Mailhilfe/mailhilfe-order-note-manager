# Mailhilfe Order Note Manager for WooCommerce

Plugin WordPress per creare, organizzare e utilizzare modelli riutilizzabili per le note degli ordini WooCommerce.

## Versione italiana

La versione 2.0.19 include la traduzione italiana completa dell’interfaccia, della guida integrata, delle domande frequenti e dei modelli dimostrativi. Per utilizzare l’interfaccia italiana, seleziona Italiano come lingua del sito o dell’utente in WordPress.

## Funzioni principali

- Modelli per note interne e note per il cliente
- Segnaposto per dati dell’ordine, del cliente, della spedizione e del pagamento
- Anteprima modificabile prima dell’aggiunta della nota
- Categorie, preferiti, ricerca e ordinamento
- Condizioni dei modelli
- Cronologia centralizzata e registro dell’elaborazione email
- Importazione ed esportazione JSON con anteprima
- Compatibilità con WooCommerce HPOS
- Permessi separati per la gestione e l’utilizzo dei modelli

## Installazione

1. Carica la cartella `mailhilfe-order-note-manager` in `/wp-content/plugins/` oppure installa il file ZIP nell’area di amministrazione di WordPress.
2. Attiva il plugin.
3. Apri **Note ordine Mailhilfe** nel menu di WordPress.
4. Crea un modello oppure installa i modelli dimostrativi.

## Traduzioni

Il repository include cataloghi verificati per tedesco standard, tedesco formale, spagnolo, francese, italiano, russo e portoghese brasiliano. Il dominio di testo è `mailhilfe-order-note-manager`.
