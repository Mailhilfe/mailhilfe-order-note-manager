# Mailhilfe Order Note Manager for WooCommerce

## 日本語版

バージョン 2.0.19 には、日本語の管理画面、組み込みヘルプ、よくある質問、デモテンプレートが含まれています。日本語表示を使用するには、WordPress のサイト言語またはユーザー言語を日本語に設定してください。

## 主な機能

- WooCommerce 注文メモテンプレートの作成と管理
- 内部メモと顧客向けメモ
- 注文、顧客、支払い、配送、商品用プレースホルダー
- カテゴリー、お気に入り、検索、並べ替え、使用回数
- JSON インポート、エクスポート、インポートプレビュー
- テンプレート条件、一元化された履歴、メール処理ログ
- テスト注文プレビュー、個人のお気に入り、最近使用したテンプレート
- WooCommerce HPOS 対応

## 安全上の注意

顧客向けメモは WooCommerce のメール通知を送信する場合があります。メモを追加する前に、プレビュー、プレースホルダー、個人情報、メモ種類を必ず確認してください。

## 翻訳ファイル

```text
languages/mailhilfe-order-note-manager-ja.po
languages/mailhilfe-order-note-manager-ja.mo
```

translate.wordpress.org へ提出する前に、日本語を母語とする翻訳者による追加確認と WordPress 日本語用語集への準拠を推奨します。
