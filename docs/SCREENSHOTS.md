# Screenshots

The repository contains documentation screenshots with example data:

1. `screenshot-1-template-editor.png` — template editor, language, conditions, placeholders and test-order preview.
2. `screenshot-2-order-screen.png` — WooCommerce order integration with editable preview.
3. `screenshot-3-history.png` — central note and email-processing history with order links.
4. `screenshot-4-settings.png` — settings and role permissions.
5. `screenshot-5-import-export.png` — JSON import preview, export and demo templates.
6. `screenshot-6-diagnostics.png` — diagnostics and developer extension status.

The screenshots use fictional data and do not contain real customer information.

For WordPress.org, copy the screenshots to the SVN repository's top-level `assets` directory. Do not place them inside the installable plugin ZIP.
