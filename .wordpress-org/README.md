# WordPress.org assets

These files are repository assets for the WordPress.org SVN `assets` directory.

They must not be copied into the installable plugin folder or release ZIP. The release build excludes this directory automatically.

Expected SVN layout:

```text
assets/
  banner-772x250.png
  banner-1544x500.png
  icon-128x128.png
  icon-256x256.png
  screenshot-1.png
  screenshot-2.png
  screenshot-3.png
  screenshot-4.png
  screenshot-5.png
  screenshot-6.png
```
