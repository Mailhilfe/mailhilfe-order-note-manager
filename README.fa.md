# مدیریت یادداشت سفارش Mailhilfe برای ووکامرس

نسخه 2.0.19 شامل رابط مدیریتی فارسی، راهنمای داخلی، پرسش‌های متداول و الگوهای نمایشی فارسی است. برای استفاده از رابط فارسی، زبان سایت یا زبان کاربر وردپرس را روی فارسی تنظیم کنید.

## امکانات

- الگوهای قابل استفاده مجدد برای یادداشت سفارش ووکامرس
- یادداشت داخلی و یادداشت مشتری
- پیش‌نمایش قابل ویرایش با داده‌های سفارش
- شرایط الگو بر اساس وضعیت سفارش، روش پرداخت، روش حمل‌ونقل، کشور صورتحساب و مبلغ سفارش
- انتخاب زبان الگو برای فروشگاه‌های چندزبانه
- برگزیده‌های شخصی و الگوهای اخیراً استفاده‌شده
- تاریخچه مرکزی ایجاد یادداشت، استفاده از الگو و پردازش ایمیل
- پیش‌نمایش سفارش آزمایشی در ویرایشگر الگو
- درون‌ریزی و برون‌بری JSON همراه با پیش‌نمایش
- مدیریت نقش‌ها و دسترسی‌ها
- سازگاری با HPOS ووکامرس
- راهنمای داخلی و پرسش‌های متداول فارسی

## نصب

1. فایل ZIP قابل نصب را دریافت کنید.
2. در وردپرس به **افزونه‌ها ← افزودن افزونه تازه ← بارگذاری افزونه** بروید.
3. فایل ZIP را بارگذاری و افزونه را فعال کنید.
4. منوی **یادداشت‌های سفارش Mailhilfe** را باز کنید.
5. یک الگو بسازید یا الگوهای نمایشی را نصب کنید.
6. یک سفارش ووکامرس را باز و از کادر الگوی یادداشت سفارش استفاده کنید.

## فایل‌های زبان فارسی

```text
languages/mailhilfe-order-note-manager-fa_IR.po
languages/mailhilfe-order-note-manager-fa_IR.mo
```

زبان فارسی وردپرس با کد `fa_IR` پشتیبانی می‌شود. پیش از ارسال ترجمه به WordPress.org، بازبینی نهایی توسط مترجم فارسی‌زبان توصیه می‌شود.

## مجوز
