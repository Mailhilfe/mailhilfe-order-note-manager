# Mailhilfe Order Note Manager for WooCommerce

Phiên bản 2.0.19 bao gồm giao diện quản trị, trợ giúp tích hợp, câu hỏi thường gặp và mẫu minh họa bằng tiếng Việt. Để dùng giao diện tiếng Việt, hãy đặt ngôn ngữ trang web hoặc ngôn ngữ người dùng WordPress thành Tiếng Việt.

## Tính năng

- Mẫu ghi chú đơn hàng WooCommerce có thể tái sử dụng
- Ghi chú nội bộ và ghi chú cho khách hàng
- Bản xem trước có thể chỉnh sửa bằng dữ liệu đơn hàng
- Điều kiện mẫu theo trạng thái đơn hàng, phương thức thanh toán, phương thức giao hàng, quốc gia thanh toán và tổng đơn hàng
- Lựa chọn ngôn ngữ mẫu cho cửa hàng đa ngôn ngữ
- Mục yêu thích cá nhân và mẫu dùng gần đây
- Lịch sử tập trung cho việc tạo ghi chú, sử dụng mẫu và xử lý email
- Xem trước bằng đơn hàng thử nghiệm trong trình soạn thảo mẫu
- Nhập và xuất JSON có bước xem trước
- Quản lý vai trò và quyền
- Tương thích WooCommerce HPOS
- Trợ giúp và câu hỏi thường gặp bằng tiếng Việt

## Cài đặt

1. Tải tệp ZIP có thể cài đặt.
2. Trong WordPress, mở **Plugin → Thêm plugin mới → Tải plugin lên**.
3. Tải tệp ZIP lên và kích hoạt plugin.
4. Mở menu **Ghi chú đơn hàng Mailhilfe**.
5. Tạo mẫu hoặc cài đặt mẫu minh họa.
6. Mở một đơn hàng WooCommerce và dùng hộp chọn mẫu ghi chú đơn hàng.

## Tệp ngôn ngữ tiếng Việt

```text
languages/mailhilfe-order-note-manager-vi.po
languages/mailhilfe-order-note-manager-vi.mo
```

Plugin hỗ trợ locale WordPress `vi`. Trước khi gửi bản dịch lên WordPress.org, nên có một người bản ngữ tiếng Việt kiểm tra lần cuối.
