# Mailhilfe Order Note Manager for WooCommerce

Verze 2.0.19 obsahuje české administrační rozhraní, vestavěnou nápovědu, časté dotazy a ukázkové šablony. Pro české rozhraní nastavte jazyk webu nebo uživatelského profilu WordPressu na Češtinu.

## Funkce

- Opakovaně použitelné šablony poznámek k objednávkám WooCommerce
- Interní poznámky a poznámky pro zákazníky
- Upravitelný náhled se skutečnými údaji objednávky
- Podmínky šablon podle stavu objednávky, platební metody, dopravy, fakturační země a hodnoty objednávky
- Výběr jazyka šablony pro vícejazyčné obchody
- Osobní oblíbené položky a naposledy použité šablony
- Centrální historie vytváření poznámek, použití šablon a zpracování e-mailů
- Náhled s testovací objednávkou v editoru šablony
- Import a export JSON s potvrzovacím náhledem
- Správa rolí a oprávnění
- Kompatibilita s WooCommerce HPOS
- Vestavěná nápověda a časté dotazy v češtině

## Instalace

1. Stáhněte instalovatelný soubor ZIP.
2. Ve WordPressu otevřete **Pluginy → Instalace pluginů → Nahrát plugin**.
3. Nahrajte ZIP a plugin aktivujte.
4. Otevřete nabídku **Poznámky k objednávkám Mailhilfe**.
5. Vytvořte šablonu nebo nainstalujte ukázkové šablony.
6. Otevřete objednávku WooCommerce a použijte výběr šablony poznámky.

## České jazykové soubory

```text
languages/mailhilfe-order-note-manager-cs_CZ.po
languages/mailhilfe-order-note-manager-cs_CZ.mo
```

Plugin podporuje WordPress locale `cs_CZ`. Před odesláním překladu na WordPress.org se doporučuje závěrečná kontrola rodilým českým mluvčím.
