# Mailhilfe Order Note Manager for WooCommerce

Плагин предоставляет повторно используемые шаблоны примечаний к заказам WooCommerce, предпросмотр заполнителей, внутренние примечания и примечания для клиентов, условия шаблонов, историю, диагностику, импорт и экспорт JSON.

## Русская локализация

Версия 2.0.19 содержит проверенный перевод интерфейса, встроенной справки, FAQ и демонстрационных шаблонов на русский язык. Для выбора русского интерфейса установите язык сайта или пользователя WordPress на русский.

## Установка

1. Загрузите ZIP-файл плагина через «Плагины → Добавить плагин → Загрузить плагин».
2. Активируйте плагин.
3. Откройте «Примечания Mailhilfe» в панели управления WordPress.
