# Mailhilfe Order Note Manager for WooCommerce

## 简体中文版本

版本 2.0.19 包含完整的简体中文界面、内置帮助、常见问题和演示模板翻译。要使用中文界面，请在 WordPress 中将站点语言或用户语言设置为简体中文。

## 主要功能

- 创建和管理 WooCommerce 订单备注模板
- 支持内部备注和客户备注
- 使用订单、客户、付款、配送和商品占位符
- 模板分类、收藏、搜索、排序和使用次数统计
- JSON 导入、导出与导入预览
- 模板条件、集中历史记录和电子邮件处理日志
- 测试订单预览、个人收藏和最近使用模板
- 兼容 WooCommerce HPOS

## 安全提示

客户备注可能触发 WooCommerce 电子邮件通知。添加备注前，请务必检查预览、占位符、个人数据和备注类型。

## 翻译说明

简体中文语言文件位于：

```text
languages/mailhilfe-order-note-manager-zh_CN.po
languages/mailhilfe-order-note-manager-zh_CN.mo
```

在将翻译提交到 translate.wordpress.org 前，建议由中文母语译者再次审核，并遵循 WordPress 简体中文术语规范。
