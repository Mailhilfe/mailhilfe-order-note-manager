# Translations

Text domain:

```text
mailhilfe-order-note-manager
```

Bundled language sources are stored in `languages/` as PO/MO files. The POT source is:

```text
languages/mailhilfe-order-note-manager.pot
```

## Adding or updating strings

1. Use WordPress internationalization functions for user-facing PHP strings.
2. Use the plugin text domain exactly as written above.
3. Update the POT file.
4. Update the affected PO files.
5. Compile MO files.
6. Test both the site locale and the individual user-profile locale.
7. Test locale variants such as `de_DE_formal` where applicable.

## Bundled locales

Arabic, Danish, German, formal German, Spanish, French, Hindi, Indonesian, Italian, Japanese, Korean, Dutch, Polish, Brazilian Portuguese, Russian, Swedish, Thai, Turkish, Ukrainian, Vietnamese and Simplified Chinese.

WordPress.org language packs may take precedence over bundled translations when the plugin is distributed through the directory.
